Luồng kết nối (ưu tiên)
WiFi → 4G (SIM7680C) → AP Mode (offline)

WiFi / 4G: gửi dữ liệu lên ThingsBoard qua MQTT mỗi 5 giây.
AP Mode: ESP32 tạo hotspot UTT-MayThiCong để truy cập dashboard nội bộ.
Ở AP Mode, cứ 30 giây tự động thử kết nối lại 4G.

Tính năng chính:
- Theo dõi giờ hoạt động: dùng ngắt (interrupt) từ cảm biến rung để xác định 
máy đang chạy hay dừng, tích lũy totalRunMs và totalIdleMs, lưu vào 
NVS (Preferences) mỗi 1 phút.
- Cảnh báo idle: máy đang nổ nhưng không di chuyển (GPS < 1 km/h).

- Pin: INA219 lấy dữ liệu và tính phần trăm pin Li-Ion theo bảng tra điện áp → hiển thị icon pin 
trên OLED.

- Web dashboard (ở chế độ AP mode): giao diện HTML có bản đồ Leaflet.js, 
thẻ nhiệt độ, trạng thái máy ON/OFF, cập nhật realtime qua /api/status.

Dữ liệu gửi lên ThingsBoard (MQTT)
engine_on, totalRunMinutes, totalIdleMinutes, latitude, longitude, temperature, power_mw
